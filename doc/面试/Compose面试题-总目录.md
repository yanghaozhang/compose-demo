## 💡 面试技巧

### 回答问题的结构：
1. **概念定义** - 简明扼要说明是什么
2. **为什么需要** - 解决什么问题
3. **如何使用** - 给出代码示例
4. **最佳实践** - 注意事项和优化建议
5. **对比分析** - 与其他方案的区别

### 示例（以"什么是状态提升"为例）：
```
1. 概念：状态提升是将状态从子组件移到父组件的模式
2. 原因：实现单一数据源、组件可重用性、便于测试
3. 示例：[给出代码]
4. 最佳实践：提升到最低公共祖先、保持组件无状态
5. 对比：与传统 View 系统的状态管理区别
```
